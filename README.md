# Persistent-Homology_PH-SAM2
PH-SAM2 combines persistent homology and SAM2 (Meta/SAM2) for fully automatic, zero-shot segmentation of knee implants in X-rays without fine-tuning. Topology-based centroids guide SAM2 prompts. Tested on 310 radiographs, it performed consistently across model sizes, with SAM2-Large best (Mean Dice = 0.9008).
