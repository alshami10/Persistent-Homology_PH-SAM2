## Project Description
PH-SAM2 combines Persistent Homology and Meta segmentation model [SAM2](https://github.com/facebookresearch/sam2) for fully automatic, zero-shot segmentation of knee implants in X-rays without fine-tuning. Topology-based centroids guide SAM2 prompts. Tested on 310 radiographs, it performed consistently across model sizes, with SAM2-Large achieving the best performance (**Mean Dice = 0.9008**).
